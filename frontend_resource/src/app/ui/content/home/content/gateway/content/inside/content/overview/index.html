  • 使用说明

  • 1. 如何访问网关？

    ①在请求的头部加入标签Eo-Gateway-Token，用于进行网关鉴权，标签的值为Gateway Token（可在网关概况页面获取）

    ②通过HTTP方式访问网关的请求地址（如http://test.gateway.com/21b1697a52f6781d901afa6a2a07154609a7be88/test/login）

  • 2. 网关返回的HTTP状态码分别是什么意思？

    400：请求的接口缺乏必填参数

    401：缺乏网关鉴权头部Eo-Gateway-Token，请在header中包含Eo-Gateway-Token标签并填入Gateway Token

    404：没有发现该网关接口，可能并未新建或者还未生效（被禁用或者等待生效中）
